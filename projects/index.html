Proyectos


Crupzi

Crupzi.com fue fundado en octubre de 2012. Es (o era) un sitio Web que recopila los descuentos de cientos de comercios. Permite encontrar descuentos a partir de múltiples filtros como categoría, centro comercial o tarjeta requerida. También permite la suscripción a comercios para recibir notificaciones cuando se ingrese una nueva promoción.

Quoffee

Quoffee.com fue lanzado en el primer trimestre de 2014. La idea es facilitar la organización de citas ("quotes") de libros, blog posts, canciones y cualquier otra cosa. Las citas se organizan en cuadernos que pueden ser públicos o privados.

Wrteen

Wrteen fue un hackathon de dos fines de semana. Necesitaba una herramienta para compartir "posts" informales. Lo hice con la idea de que cualquier persona debería poder contar con una herramienta para escribir cosas en Internet sin necesidad de tener un website.
